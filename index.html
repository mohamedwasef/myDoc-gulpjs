First Step - Installation

  • Download Node JS (From NodeJS ) Then use NPM to start using GulpJS or use command line
  • Install Gulp Globaly in your system: sudo install -g gulp
  • Install Gulp in your project, Map to the root directory using - cd to install it directly on your local project sudo npm install --save-dev gulp
    You will find node-modules folder installed on your project files
  • Make the first default task by creating a new folder inside on the root called gulpfile.js
    var gulp = require('gulp'); // require node package
    gulp.task('default',function(){
    console.log('Hello World'); // what will run here
    });
    Run using gulp

Second Step - minify

  • To minify the files Install (Gulp Uglify) from plugins use this command sudo npm install --save-dev gulp-uglify
    It will install it's files inside node-module folder
    Define Uglify like the first step
    var gulp = require('gulp'), // require gulp node package
    var uglify = require('gulp-uglify'); // require uglify
    gulp.task('default',function(){
    gulp.src('js/*.js') // run command in particulur directory
    .pipe(uglify())
    .pipe(gulp.dest('minjs'));
    });
    Run using gulp

Third Step - Minify

  • Scripts Gulp use this command
    var gulp = require('gulp'), // require gulp node package
    var uglify = require('gulp-uglify'); // require uglify

    gulp.task('scripts',function(){
    gulp.src('js/*.js') // run command in particulur directory
    .pipe(uglify())
    .pipe(gulp.dest(minjs));
    });
    Run using gulp scripts to run scripts task only
    Or use gulp.task('default',['scripts']); Callback function
  • To run Multiple Task - Add another Task
    gulp.task('styles',function(){
    console.log('Run Styles');
    });
    Use gulp.task('default',['scripts','styles']); To make callback function to run multi task
    Run using gulp to run default function with callback

Fourth Step - Watch

  • Install & Compile SASS
    sudo npm install --save-dev gulp-ruby-sass
    Require SASS
    var sass = require('gulp-ruby-sass'); // require sass
    Add SASS Task
    gulp.task('sass', function () {
    return sass('scss/**/*.scss')
    .on('error', function (err) { console.log(err.message); })
    .pipe(gulp.dest('css/'))
    });
    });
    we can put some options like
    .pipe(sass({
    style: 'compressed'
    }))
    Watch Task - To watch JS Process
    gulp.task('watch',function(){
    gulp.watch('scss/**/*.scss',['sass']); // Any sass file
    });
    Run gulp sass
    Use gulp.task('default',['scripts','sass','watch']); To make callback function to run multi task

Fifth Step - Watch

  • Watch Task - To watch JS Process
    gulp.task('watch',function(){
    gulp.watch('js/*.js',['scripts']); // Any js file
    });
    Run gulp watch
    Use gulp.task('default',['scripts','styles','watch']); To make callback function to run multi task

Sixth Step - Plumber

  • Plumber - It helps to show Issues in CMD
    sudo npm install --save-dev gulp-plumber
    Define Plumber
    var plumber = require('gulp-plumber');
    Edit any task
    gulp.task('sass',function(){
    gulp.src('scss/**/*.scss') //watch particular directory
    .pipe(sass())
    .pipe(plumber())
    .pipe(gulp.dest('css/')) //save output in this folder
    });
    Make any typing mistake in sass file to check the problem report on CMD
    Use gulp.task('default',['scripts','sass','watch']); To make callback function to run multi task

Seventh Step - Gulp Connect

  • Live Reload- Install LiveReload Extention for browser
    Then Install gulp-connect
    npm install gulp-connect
    Define or Require Package
    connect = require('gulp-connect');
    Write the connect inside sass
    Add function after sass
    gulp.task('connect', function() {
    connect.server({
    livereload: true
    });
    });

    Write this to run the code
    gulp.task('default', ['scripts','styles','watch','connect']);

Eighth Step - LiveReload

  • Gulp Connect
    sudo npm install --save-dev gulp-livereload
    Then Install Gulp Package for LiveReload
    Define or Require Package
    livereload = require('gulp-livereload'), // require gulp livereload
    Call the Livereload inside watch
    Use var server = livereload();
    Inside SASS at the last add livereload in pipe after dest
    .pipe(livereload()); Then click the extention to start livereload with your webpage

Nineth Step - IMG Compression

  • Image Compression- Install Image Compression
    sudo npm install --save-dev gulp-imagemin
    Define and Require
    imagemin = require('gulp-imagemin'), // require gulp imagemin
    Add Task to image compression
    gulp.task('image',function(){
    gulp.src('img/*') //watch particular directory
    .pipe(imagemin())
    .pipe(gulp.dest('build/img')) //save output in this folder
    });
    run using
    gulp image

Teenth Step - Prefixing

  • Auto Browser Prefixing - Instillation
    sudo npm install gulp-autoprefixer
    Define and Require
    prefix = require('gulp-autoprefixer'), // require gulp prefixer
    Add Task to image compression
    gulp.task('sass',function(){
    gulp.src('scss/**/*.scss') //watch particular directory
    .pipe(sass())
    .on('error',console.error.bind(console))
    .pipe(prefix('last 2 versions'))
    .pipe(gulp.dest('css/')) //save output in this folder
    });
    run using
    gulp sass
